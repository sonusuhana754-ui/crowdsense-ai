# 📌 Frontend Overview

The CrowdSense AI frontend is a futuristic command-center dashboard designed to visualize:

- 🎥 Live surveillance feeds
- 📊 Crowd density analytics
- ⚠️ Real-time risk assessment
- 🧠 AI-generated incident intelligence
- 📡 Superintendent command system
- 🚪 Exit flow management
- 📜 Live alert logs



# 🎨 Frontend Design Features

## ✅ Premium Command Center UI

- Cyberpunk-inspired dark theme
- Neon glow effects
- Live operational aesthetic
- Real-time animated indicators
- Responsive grid layout

---

## ✅ Multi Camera Surveillance Grid

### Top Cameras
- Live webcam surveillance feeds
- Real-time monitoring

### Bottom Cameras
- Video upload analysis
- Crowd footage processing

---

## ✅ AI Risk Dashboard

Displays:

- Composite Risk Score
- Incident Severity Level
- Crowd Count Estimation
- Density Calculation
- Critical Threshold Timer

---

## ✅ Dual AI Intelligence Panels

### Junior Analyst
Initial AI interpretation of surveillance footage.

### Senior Critic Model
Advanced refined analysis with:
- Occlusion correction
- Risk escalation
- Tactical recommendations
- Command generation

---

## ✅ Superintendent Operations Panel

Includes:
- Radio commands
- Exit management
- Crowd flow control
- Emergency instructions
- Public announcement scripts

---

## ✅ Live Alert Logging

Real-time event stream with:
- Timestamps
- Severity labels
- AI detections
- System events
